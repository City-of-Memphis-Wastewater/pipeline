◆ FieldId

typedef int eds::live::FieldId

Identifier of process point field.

Fields of process points have both string names and numeric identifiers (FieldIds). Referencing point names using identifiers is more efficient.

Use LiveClient::fieldIdFromName() and LiveClient::fieldIdFromWDPFName() methods to convert field names to identifiers.

◆ PointGroups

typedef std::vector<bool> eds::live::PointGroups

Bit vector of point groups.

Point groups are identified by a 0-based index, with the admin group having index 0.

Currently 256 groups are defined, although this number might increase in future versions of EDS.
