Calling report functions

The archive client from EDS API can call report functions in Archive Server.

A report function takes a range of point samples from EDS archive and aggregates them into a single value and quality.

Every report function requires several parameters. The exact number of parameters depends on the function:

  • In general, all functions expect to receive exactly one point identifier.
  • Most functions will also require two timestamps - the beginning and end of the archive range to process.
    The only exception to this rule is the VALUE function, which expects only one timestamp.
  • Some functions will require additional numeric parameters (for example a limit or threshold which must be crossed).

For a detailed list of report functions and their parameters, please refer to the EDS User Manual.

To improve performance, multiple report function calls can be batched together and executed simultaneously.

Defining a report function call

Before a report function can be executed on Archive Server, the call must be registered in eds::arch::ArchClient. Note that the client should be already initialized.

The example below constructs two function calls - one for the MAX_VALUE function, the other for AVG.

The maxValueSid and avgSid variables used in this example should contain the SID identifiers of queried points. A SID can be obtained from an IESS using eds::live::LiveClient. Please see the Process point identifiers page for more information.

try
{
time_t now = time(nullptr);
// Pointers are wrapped in unique_ptr to automatically release objects
// in case of exception.
// Calculate value maximum for point with SID <maxValueSid> over last 24 hours.
std::unique_ptr<ReportFunction> maxValueCall(client->getReportFunction("MAX_VALUE"));
maxValueCall->pushPointParam(maxValueSid);
maxValueCall->pushTimestampParam(now - 24 * 3600);
maxValueCall->pushTimestampParam(now);
// Calculate average for vallue point with SID <avgSid> over last 24 hours.
// Shades will have priority over values from archive.
std::unique_ptr<ReportFunction> avgCall(client->getReportFunction("AVG"));
avgCall->pushPointParam(avgSid, 0xFF, ShadeMode_PreferShade);
avgCall->pushTimestampParam(now - 24 * 3600);
avgCall->pushTimestampParam(now);
// Functions will be executed here in the next step.
}
catch (const ArchClientError& exc)
{
// Handle the error (display an error message, etc).
printf(
"Couldn't create definition of a report function call (%s)\n",
exc.what());
}

Executing report functions

The function calls defined in the previous step can be now registered for execution. Pass each eds::arch::ReportFunction* to the eds::arch::ArchClient::addQuery method and remember the returned identifier. It will be used to fetch the values returned by report functions.

When all function calls are registered in the client, call the eds::arch::ArchClient::executeQueries method. It will block until all values of requested report functions are available.

try
{
// Function definition from the previous example omitted for brevity.
// Remember the returned function call identifiers.
int maxValueQueryId = client->addQuery(maxValueCall.get());
int avgQueryId = client->addQuery(avgCall.get());
// This function will block until reports are ready.
client->executeQueries();
// Function call results will be fetched here in the next step.
}
catch (const ArchClientError& exc)
{
// Handle the error (display an error message, etc).
printf(
"Couldn't execute report function calls (%s)\n",
exc.what());
}

Fetching results from report function calls

To access the result of a report function call, use the eds::arch::ArchClient::getResponse method.

Should a report function call fail, the returned quality value will be equal to eds::Quality_None.

try
{
// Function call identifiers from the previous example.
int maxValueQueryId = ...;
int avgQueryId = ...;
double maxValue = client->getResponse(maxValueQueryId, &quality);
printf("MAX_VALUE = %f%c\n", maxValue, quality);
double avgValue = client->getResponse(avgQueryId, &quality);
printf("AVG = %f%c\n", avgValue, quality);
}
catch (const ArchClientError& exc)
{
// Handle the error (display an error message, etc).
printf(
"Couldn't fetch value from a report function call (%s)\n",
exc.what());
}
